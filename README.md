# analisis-violencia-genero-medellin-2018-2023-
 Análisis de violencia de género en Medellín (2017–2024)

## Descripción
Este proyecto analiza los casos de violencia de género registrados en Medellín utilizando técnicas de análisis de datos y machine learning.

## Objetivo
Identificar patrones territoriales y sociodemográficos asociados a los casos reportados.

## Metodología
- Limpieza de datos
- Análisis exploratorio (EDA)
- Reducción de dimensionalidad con PCA
- Clustering con K-means
- Prueba estadística Chi-cuadrado

## Herramientas
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Power BI

## Resultados
Se identificaron patrones territoriales y agrupaciones de comunas con características similares en los casos reportados.

## Hallazgos principales

A partir del análisis exploratorio y las técnicas de minería de datos aplicadas, se identificaron varios patrones relevantes en los casos de violencia de género en Medellín:

• Se observaron diferencias territoriales en la concentración de casos, con algunas comunas presentando mayor incidencia que otras.

• El análisis de reducción de dimensionalidad mediante PCA permitió identificar las variables con mayor influencia en la caracterización de los casos reportados.

• El algoritmo de clustering K-means permitió agrupar comunas con características similares en términos de variables sociodemográficas y tipos de violencia reportados.

• Se identificaron perfiles diferenciados de víctimas según variables como edad y contexto socioeconómico.

• Las pruebas estadísticas (Chi-cuadrado) evidenciaron asociaciones significativas entre algunas variables categóricas del dataset.

Estos hallazgos permiten comprender mejor la estructura territorial y sociodemográfica de la violencia de género en Medellín, aportando información útil para el diseño de políticas públicas y estrategias de prevención.
